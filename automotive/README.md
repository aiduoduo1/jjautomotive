# Android Automotive 开发环境

这是一个针对Android Automotive OS开发的基础项目环境配置。

## 环境配置

### 构建工具版本
- Gradle: 8.2
- Android Gradle Plugin (AGP): 8.1.2
- Kotlin: 1.8.20

### SDK 配置
- compileSdk: 33 (Android 13)
- minSdk: 29 (Android 10/Q)
- targetSdk: 33 (Android 13)

### Java 环境
- Java版本: 17
- Kotlin JVM Target: 17
